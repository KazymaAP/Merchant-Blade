## 🎮 Как играть

### Core Loop (Основной цикл):
1. **Торговля** - покупайте дешёво в одних городах, продавайте дорого в других
2. **Путешествие** - при путешествии может быть бой (30% шанс)
3. **Репутация** - растёт с продажами, влияет на цены
4. **Титулы** - получайте титулы за репутацию и капитал, они снижают налоги
5. **Охрана** - нанимайте охранников в таверне для защиты от боев

### Управление деньгами:
- Все цены указаны в медяках (м.)
- 1 серебро (с.) = 100 медяков
- 1 золото (з.) = 100 серебра = 10000 медяков

## 🔧 Новые системы в v0.2.0

### TitleSystem (Система титулов)
- **Странник** (репутация 0-49) - стандартные налоги
- **Торговец** (репутация 50+, капитал 100 серебра) - налог 95% от базового
- **Купец** (репутация 100+, капитал 500 серебра) - налог 90%
- **Купец-князь** (репутация 200+, капитал 5000 серебра) - налог 80%

### GuardSystem (Система охраны)
- **Лучник** - 50 медяков за поездку, -5% шанс боя
- **Мечник** - 75 медяков, -8% шанс боя
- **Рыцарь** - 150 медяков, -15% шанс боя
- Мораль охраны падает при поражениях

### Интеграция с боями
- При путешествии 30% вероятность встречи с разбойниками
- Охрана снижает вероятность боя
- События (война) повышают вероятность
- При победе получаете лут
- При поражении теряете товары и репутацию
