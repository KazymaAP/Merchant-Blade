## 🚀 РАСШИРЕННЫЕ ФУНКЦИИ

### Новая Экономическая Модель
- Покупка/продажа теперь работает через арбитраж (разница цен между городами)
- Налоги применяются корректно
- Титулы дают скидки на налоги (-5% до -20%)

### Система Репутации (3 уровня)
- **Неизвестный** (0-19) - цены ×1.10
- **Уважаемый** (20-49) - цены ×1.05
- **Известный** (50-74) - цены ×1.00
- **Легендарный** (75+) - цены ×0.95

### Система Титулов (4 ранга)
- **Странник** - репутация 0+, капитал 0+
- **Торговец** - репутация 50+, капитал 100 серебра
- **Купец** - репутация 100+, капитал 500 серебра
- **Купец-князь** - репутация 200+, капитал 5000 серебра

### Система Охраны (3 типа)
- **Лучник** - 50м за поездку, -5% боя
- **Мечник** - 75м за поездку, -8% боя
- **Рыцарь** - 150м за поездку, -15% боя

### Система Событий (5 типов)
- **Голод** - зерно ×1.5
- **Урожай** - зерно ×0.7
- **Ярмарка** - все товары ×1.3
- **Чума** - товары портятся быстрее
- **Война** - налоги ×1.5, боя ×1.5

## 🔍 КОД ПРИМЕРЫ

### Правильная логика торговли:
```gdscript
# Купили товар за 100м в городе А
GameManager.add_item(good_id, 100, spoil_day)

# Переехали в город Б
GameManager.travel_to_city(1)

# Продали товар за 150м в городе Б
EconomyManager.sell_good(good_id, 100, 1)
# Налог 5%: 150 * 100 * 0.05 = 750м налог
# Чистый доход: 15000 - 750 = 14250м
# Прибыль: 14250 - 10000 = 4250м ✅
```

### Система репутации с титулами:
```gdscript
# Продаём товары, получаем репутацию
EconomyManager.sell_good(good_id, quantity, city)
# → ReputationSystem.add_reputation(rep_gain)
# → TitleSystem.update_title()
# → Если титул изменился, налог уменьшается

# Налог с модификаторами:
var tax_rate = ConfigManager.get_tax_rate()  # 0.05
tax_rate *= TitleSystem.get_tax_modifier()    # ×0.90 для Купца
# Итоговый налог: 5% × 90% = 4.5%
```

## 🎮 КАК ИГРАТЬ С НОВЫМИ СИСТЕМАМИ

### Стратегия 1: Становление Купцом
1. Начните как Странник (налог 100%)
2. Торгуйте 20+ раз для репутации
3. Достигните 50+ репутации → станьте Торговцем (налог 95%)
4. Накопите капитал 500 серебра
5. Достигните Купца (налог 90%)

### Стратегия 2: Безопасная торговля
1. Начните с охраной (Лучник за 50м)
2. Боевых встреч станет на 5% меньше
3. Накопите капитал
4. Нанимайте лучших охранников (Рыцарь за 150м)
5. Боевых встреч станет на 15% меньше

### Стратегия 3: Квест-ориентированная
1. Принимайте квесты в таверне
2. Каждый квест даёт 1000-2500м + репутацию
3. Быстрее поднимайте репутацию
4. Получайте скидки на налоги раньше
